## Current meta (mid-2026)

**Best solo money:** Cayo Perico on repeat, padded during cooldown with the **Agency payphone+security loop** and **daily collectibles**, all while businesses (Acid Lab, Bunker, Nightclub, Cocaine) produce in the background — **mansion-boosted** if you own one.

**Best passive stack:** Acid Lab (best solo passive) + Nightclub (techs on Coke/Meth/Cash + safe) + Bunker, with Money Fronts boosting Counterfeit/Weed if you run them. Sell only on 2x weeks.

**Biggest free money:** the daily reset loop (~$300k/day) + Daily Objective streaks + Lucky Wheel podium car. New players: Cluckin' Bell Farm Raid ($500k free) and the free Acid Lab.

**What the mansion changed:** it doesn't add a new earner but its 24h +66% production boost (rotated daily across businesses) and raid-free security meaningfully raise passive throughput for whales who already own the businesses. It's an endgame amplifier, not a starting point.

## How to keep this current

- The **weekly event** (2x/3x, podium, discounts) changes every Thursday 07:00 UTC — always pull it live via the MCP `gta-weekly-bonuses` tool rather than guessing.
- New DLCs land roughly twice a year (summer + winter). When one drops, fetch the **Rockstar Newswire** announcement and the **gtabase patch notes** to update `money-methods.md`, `businesses.md`, and `properties-catalog.md`.
- Treat specific GTA$ figures as consensus snapshots; exact values drift with patches and bonus weeks.
